About

This website uses only html and css. The source code is available on github. This is inspired by this website. Although I do care about the loading speed, security and privacy of this, I would be lying if I didn't also mention that practicality the foremost reason of the website being the way it is. Plain CSS is plenty for a website like this, and this allows me to write posts in simple HTML, or even markdown if I want to without bothering with github actions or frameworks that generate 100 auxiliary files to make a decent website.

That being said, I would like for the web to be a bit more like this website, and a bit less like all the websites that ask about your cookie preferences using sentences like “They do not ask for you consent”.
